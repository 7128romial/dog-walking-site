# WalkMe - מערכת ניהול התנדבויות

## 📋 אודות הפרויקט

פרויקט סטודנטי לבניית אתר אינטרנט לעמותת "חברים על ארבע" - עמותה לניהול התנדבויות עם כלבים.

האתר נבנה כחלק מלימודי פיתוח אתרים, ומציג שימוש בטכנולוגיות מודרניות כמו HTML5, CSS3, JavaScript, Bootstrap ו-PHP.

## 🎯 מטרות הפרויקט

- יצירת ממשק ידידותי למתנדבים
- הצגת מידע על העמותה ועל הכלבים
- מערכת הרשמה להתנדבויות
- עיצוב רספונסיבי ומקצועי

## 🛠️ טכנולוגיות בשימוש

- **Frontend**: HTML5, CSS3, JavaScript (jQuery), Bootstrap 5
- **Backend**: PHP, MySQL
- **כלים**: Git, VS Code
- **עיצוב**: RTL (עברית), גופנים מותאמים

## 📁 מבנה הפרויקט

```
WalkMe/
├── index.html              # דף הבית
├── includes/
│   ├── about.html          # אודות העמותה
│   ├── dogs.html           # רשימת כלבים

│   ├── registration.html   # הרשמה להתנדבות
│   ├── dog-details.html    # פרטי כלב ספציפי
│   ├── save_volunteer.php  # עיבוד הרשמה (PHP)
│   └── db_connect.php      # חיבור למסד נתונים
├── css/
│   └── style.css           # עיצוב מותאם אישית
├── js/
│   ├── main.js             # JavaScript כללי
│   ├── about.js            # JavaScript לדף אודות
│   ├── dogs.js             # JavaScript לדף כלבים
│   ├── dog-details.js      # JavaScript לפרטי כלב

├── images/                 # תמונות וגרפיקה
└── README.md              # קובץ זה
```

## 🚀 התקנה והרצה

1. הורד את הקבצים
2. פתח את `index.html` בדפדפן
3. לשרת מקומי: `python -m http.server 8000`
